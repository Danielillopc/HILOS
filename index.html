¿What is HILOS?

HILOS is a software created for schools, universities or training companies, which can be used as an online learning tool available 24-7-365. His idea is simple, a HILO (Highly Interactive Learning Digital Object) is an online platform to create a virtual learning environments for students or learners, with a powerful user interface with drop-down windows that facilitate learning using maps, videos, infograpics, e-books, excercies,etc.

The advantage from the teacher's point of view is that the time needed to develop a HILO is considerably less than designing an online course on platforms such as classroom or moodle.

The advantage for students or learners is that it does not matter if they are readers, visual, auditory or kinesthetic, the platform provides resources for all learning styles, which makes it a personalized learning platform.
